# Expense Tracker

an application built in python used to pull latest events of a github user

# Description
simple app that when used, pulls latest changes and events that the user performed in github repos


## Installation

pull the repo, install requirements.txt and use the "github.py" located in 'src/github-activity/' directory

## Usage

```python
# list help options to start using the application
github.py --help
```
